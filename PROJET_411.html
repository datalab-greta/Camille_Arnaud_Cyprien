CHAMPS SELECTIONNES POUR L'ANALYSE¶

dans la table POSTGRES :¶

- ID : numéro identifiant la discussion.¶

- COURSE_ID : nom identifiant le nom du cours associé à la discussion.¶

- DATE_UPDATED : date de mise à jour, la dernière interaction dans la discussion.¶

- USERNAME : noms des apprenants.¶

- BODY : présentation des apprenants.¶

- COURSEWARE_TITLE : nom de la discussion.¶

- DATE_CREATED : date où la discussion a été créee.¶
